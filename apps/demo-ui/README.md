# Hotel Booking Demo UI

Thin React + Vite + TypeScript demo client for the hotelbooking backend platform.

The UI is intentionally lightweight. Its purpose is to demonstrate backend flows:

- hotel catalog
- room availability
- booking saga creation
- current user's bookings
- audit timeline
- booking cancellation
- inventory admin operations

## Current state

This first version contains only the UI shell.

Backend API clients will be added in the next steps.

## Run locally

```bash
npm install
npm run dev
